# 品花社

妹子壁纸类应用，减少寻找壁纸的烦恼。

具备功能：

- 一键设为壁纸轻松快捷
- 一键分享图片或者下载到本地
- 喜欢的图片直接收藏，下次不再错过

## 数据来源

感谢知乎让我知晓[品花社](http://m36.com),本应用所有图片均来自上述网站
